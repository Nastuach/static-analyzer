# Статистический анализатор данных

## Описание проекта

Программа на Python для первичной статистической обработки и визуализации выборки данных. Проект представляет собой Jupyter Notebook с полным анализом одномерного статистического распределения.

## Функциональные возможности

### 1. Построение интервального ряда
- Определение количества интервалов по формуле Стерджесса
- Расчет ширины интервала
- Подсчет частот и относительных частот

### 2. Дискретный ряд распределения
- Вычисление середин интервалов
- Расчет накопленных частот
- Определение накопленных относительных частот

### 3. Визуализация
- Гистограмма относительных частот
- Полигон относительных частот
- Кумулята
- Эмпирическая функция распределения

### 4. Числовые характеристики
- Мода интервального ряда
- Медиана
- Выборочная средняя
- Выборочная дисперсия
- Среднее квадратическое отклонение
- Начальные и центральные моменты

### 5. Коэффициенты
- Коэффициент вариации
- Коэффициент асимметрии
- Коэффициент эксцесса

### 6. Проверка гипотезы о нормальном распределении
- Построение теоретической нормальной кривой
- Сравнение эмпирического и теоретического распределений
- Критерий согласия Пирсона (χ²)

### 7. Доверительные интервалы
- Доверительный интервал для математического ожидания
- Доверительный интервал для среднего квадратического отклонения

Формулы
Число интервалов (Стерджесс): k = 3.322·lg(n) + 1

Ширина интервала: h = (x_max - x_min) / k

Мода: M₀ = x₀ + h·(n₂ - n₁) / ((n₂ - n₁) + (n₂ - n₃))

Медиана: Mₑ = xₑ + h·(n/2 - Sₘ₋₁) / nₑ

Доверительный интервал для MX: x̄ ± t·σ/√n

Критерий Пирсона: χ² = Σ (nᵢ - nᵢ')² / nᵢ'
